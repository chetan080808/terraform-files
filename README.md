# terraform-files
All the terraform files that i practiced during terraform course is updated here


This repository contains Terraform configurations for provisioning AWS infrastructure as part of a learning course. It uses a local Terraform module to create core resources (e.g., S3, EC2, dynamoDB) and demonstrates common workflows such as initialization, validation, planning, applying, and pushing code to GitHub.

Also it has coverved the topics like
-secure statefile management
-workspace for isolation
-remote backend using s3 and dynamoDB
